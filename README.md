# 🚴 Bike Purchase Behaviour Analysis

## 📌 Project Overview

This project focuses on analyzing customer demographics and behavioral patterns to understand **what factors influence bike purchases**. Using Excel-based data analytics techniques, the project transforms raw data into meaningful insights through **data cleaning, transformation, pivot analysis, and dashboard visualization**.

The ultimate goal is to help businesses:

* Identify target customers
* Understand purchasing patterns
* Make data-driven marketing decisions

---

## 🎯 Problem Statement

Businesses often struggle to identify **which customers are most likely to purchase bikes**. Without structured analysis, marketing efforts can be inefficient and costly.

This project answers key questions:

* Which demographic groups are more likely to buy bikes?
* How does income affect purchasing decisions?
* Does commute distance influence buying behavior?
* What role do age, occupation, and family size play?

---

## 📂 Dataset Description

The dataset contains customer demographic and lifestyle information.

### 🔑 Features:

* **ID** – Unique customer identifier
* **Marital Status** – Married / Single
* **Gender** – Male / Female
* **Income** – Annual income
* **Children** – Number of children
* **Education** – Education level
* **Occupation** – Job category
* **Home Owner** – Owns house or not
* **Cars** – Number of cars owned
* **Commute Distance** – Distance to workplace
* **Region** – Geographic region
* **Age** – Customer age
* **Purchased Bike** – Target variable (Yes/No)

## 🧠 Project Approach

### 1️⃣ Data Understanding

* Reviewed dataset structure and variables
* Identified categorical and numerical features
* Defined the target variable: **Purchased Bike**

---

### 2️⃣ Data Cleaning & Preparation (Working Sheet)

Performed transformations to improve readability and usability:

* Converted:

  * `M / S → Married / Single`
  * `M / F → Male / Female`
* Created **Age Brackets**:

  * Young
  * Middle Age
  * Old
* Standardized inconsistent values
* Ensured data integrity for analysis

---

### 3️⃣ Feature Engineering

* Added **Age Brackets** column to group customers
* Structured data for better segmentation
* Improved categorical clarity for pivot analysis

---

### 4️⃣ Data Analysis using Pivot Tables

Built multiple pivot tables to analyze relationships between variables:

#### Key Analyses:

* Income vs Bike Purchase
* Age Brackets vs Purchase Behavior
* Commute Distance vs Purchase
* Region-wise purchase trends
* Occupation impact on buying decisions

---

### 5️⃣ Dashboard Creation

Developed an interactive Excel dashboard to visualize insights:

#### 📊 Dashboard Highlights:

* Purchase behavior by **Income**
* Trends across **Age Groups**
* Impact of **Commute Distance**
* Regional distribution of buyers
* Filters for dynamic exploration

---

## 📈 Key Insights

### 💰 Income Influence

* Customers with **moderate to high income** are more likely to purchase bikes.

### 👨‍👩‍👧 Demographics

* Middle-aged individuals show higher purchase rates.
* Married individuals tend to purchase more bikes.

### 🚗 Lifestyle Factors

* Customers with **shorter commute distances** are more inclined to buy bikes.
* Lower number of cars correlates with higher bike purchases.

### 🌍 Regional Trends

* Certain regions show stronger adoption, indicating potential target markets.
